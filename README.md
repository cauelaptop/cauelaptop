# Seja bem vindo ao meu perfil
Olá! Eu sou o Cauê Shishido 

Sou alguém que gosta bastante de computação e tenho o objetivo de me tornar um cientista da computação no futuro. Gosto muito de aprender sobre tecnologia, programação e entender como as coisas funcionam “por trás das telas”.

Uma das formas que mais gosto de aprender é criando pequenos projetos, seja por curiosidade, diversão ou para testar ideias novas. Acredito que fazer projetos práticos ajuda muito a aprender e entender melhor os conceitos da computação.

Gosto bastante de criar projetos usando a linguagem de pseudocódigo Portugol, principalmente para treinar lógica de programação e experimentar ideias de sistemas e programas. Também pretendo voltar a usar Python, uma linguagem que gosto bastante.

Atualmente estou aprendendo C e Java, explorando seus conceitos e tentando aprender cada vez mais sobre programação. Quero continuar estudando, descobrir novas tecnologias e aprender muito mais no futuro.

Uma coisa que também acho muito interessante é a ideia de tentar criar uma linguagem de programação do zero. Tenho bastante curiosidade sobre como linguagens funcionam internamente e como elas são construídas.

No geral, gosto de computação, de aprender coisas novas e de transformar curiosidade em pequenos projetos
